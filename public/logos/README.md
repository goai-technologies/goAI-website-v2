## Logo Requirements:
- Format: PNG with transparent background (preferred) or white background
- Size: Recommended 400x200px or similar aspect ratio
- All logos should be clearly visible and properly sized
- File names should match exactly as listed above

## Usage:
The logos will be displayed in a scrolling animation from right to left on the home page and other pages with the "Trusted By Companies" section.
